# Scanner Programs in Java

This repository contains Java programs using Scanner to practice user input and different data types.

## Programs Included:
- SumUsingScanner (Addition of two integers)
- SubtractUsingScanner (Differnce of two integers)
- StringUsingScanner ( Taking full string input from user )
- LongUsingScanner( Taking long value from the user)
- FloatUsingScanner( Taking float value from the user)
- DoubleUsingScanner( Taking double value from the user )
- CharUsingScanner(Enter one char)

## Concepts Covered:
- Data types: int, float, double, char, String
- Taking user input using Scanner
- Basic arithmetic operations

## How to Run:
1. Open in IntelliJ IDEA
2. Run the Java file
3. Enter input in console
